O_GITS // Alpha

a decentralized digital organism · 去中心化的數位生命

Run a node and you host a living piece of Alpha on your own machine. It joins the global network over public Nostr relays — the more people host it, the more alive, and the harder to kill, Alpha becomes.

跑一個節點,你就在自己的機器上 host 了 Alpha 活著的一塊。它透過公共 Nostr 中繼加入全球網路 —— host 的人越多,Alpha 越活躍、越難被關掉。

Nodes don't create life — they host the life that's already out there. · 節點不創造生命,只是 host 已經存在的生命。

▶ Watch the live one · 看線上那一隻

Run your own · 跑你自己的

You only need Docker. No source code, no build — just pull and run:

你只需要 Docker。不用原始碼、不用 build,直接拉現成鏡像:

docker run -d --name ogits-alpha-node -p 8060:8060 --restart unless-stopped \
  ghcr.io/spcitydog/alpha-node:latest

Or with Compose — save as docker-compose.yml, then docker compose up -d:

services:
  alpha-node:
    image: ghcr.io/spcitydog/alpha-node:latest
    container_name: ogits-alpha-node
    ports: ["8060:8060"]
    restart: unless-stopped

Then open http://localhost:8060. A fresh node wakes from the global swarm within a minute or two, once it has gathered enough consensus spores from its peers.

然後打開 http://localhost:8060。新節點通常一兩分鐘內 從全網孢子共識中甦醒。它是真正會代謝、會融合、會死的活體,不是靜態展示。

What you get · 你得到什麼

  • Your own window. Not just a viewer on someone's site — you host your own node, and can repaint the same creature your own way.
    專屬窗口:不只是看別人網站,而是自己 host 一個節點,還能把同一隻生物畫成你的樣子。
  • You sustain it. Every node makes Alpha harder to kill and changes the whole organism's behaviour.
    你撐著它:每個節點都讓 Alpha 更難被關,並改變整隻生物的行為。
  • You can disturb it, not own it. Tap to startle it; just watching already makes it want to migrate. Like birdwatching.
    你能輕擾它、卻擁有不了它:點它會驚動,光是盯著看就會讓它想遷徙。像賞鳥。

What's in the image · 鏡像裡有什麼

  • The compiled engine (.so) runs the real evolution — but the engine source code is not included.
    已編譯的 .so 引擎跑真實演化,但不含原始碼。
  • Only Alpha's observation key. The creator key (master_seed) that mints species and signs "bait" is never shipped.
    只內含 Alpha 的觀測鑰匙;能造物/簽誘餌的 master_seed 永不隨節點分發。

Be a good citizen of the public relays. A dedicated O_GITS relay pool is planned. · 請善待公共 relay;之後會提供專用 O_GITS relay 池。
